# HEMS Frontend

React UI for Home Energy Management System

## Setup

```bash
npm install
npm run dev
```

Frontend runs on `http://localhost:5173`

## Folder Structure

- `src/components/` - Reusable UI components
- `src/pages/` - Page components (Dashboard, Devices, Energy)
- `src/services/` - API integration layer
- `src/context/` - State management
- `src/utils/` - Utility functions

## Available Pages

- **Dashboard** - Energy stats and charts
- **Devices** - Device management and CRUD
- **Energy Usage** - Energy consumption logging and tracking
